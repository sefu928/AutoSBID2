========================
READ BEFORE USE
========================

By default, AutoSBID selects the maximum amount of hours possible each day. If you decide to use AutoSBID on startup and you are absent from work, keep in mind that you must modify MANUALLY the day you were partially or completely absent.


========================
Setup guide
========================

To setup AutoSBID you only need to set three values in the data.json file:

-"user": your sBid username
-"password": your sBid password
-"task": the task you want AutoSBID to select in a number from 1 to 3. The value is 1 by default.


========================
Task selection guide
========================

Only one of three tasks can be selected for AutoSBID to fill:

1 selects "1.5. Utilització d'entorns de desenvolupament per a editar, depurar, provar i documentar codi, a més de generar executables".
2 selects "3.1. Manipulació de material multimèdia en múltiples formats, analitzant i utilitzant llibreries específiques per a la conversió entre formats i per a la integració de continguts multimèdia en una aplicació".
3 selects "4.3. Creació de tutorials i manuals d'usuari, d'instal·lació i de configuració de les aplicacions desenvolupades".


========================
Execute on startup guide
========================

AutoSBID has been designed to be executed on startup.

1. Open run with the Windows key + "R" key
2. Write "shell:startup" and press the Enter key
3. Copy the AutoSBID shortcut on this folder and paste it there

Now, when you start the computer, AutoSBID will run automatically.
